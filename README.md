# Application Stack
```
Stack is built with a Vue3 Front end, MongooseDB, Express Server, and Node Backend
```

## Project setup
```
npm install - Requires Node vers >16
```

### Compiles and hot-reloads for development
```
Navigate to /client 
npm run start:client

&&

Navigate to /server
npm run start:server
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
